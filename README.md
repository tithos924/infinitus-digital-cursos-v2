# Infinitus Digital Cursos

Plataforma de cursos online (estilo Hotmart/Kiwify) — monorepo com:

- `apps/web` — Next.js 14 (App Router, TypeScript, Tailwind, design system Infinitus)
- `apps/api` — NestJS (REST API, JWT auth, Prisma)
- `packages/database` — Schema Prisma + PostgreSQL

Ver `DEPLOY.md` para instruções de deploy (Vercel + Neon/Supabase Postgres).
